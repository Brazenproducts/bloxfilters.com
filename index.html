How BLOX™ Works

The BLOX™ system separates the filter housing from the filter media. The 100% recycled non-fibrous corrugate housing forms the structural container that slides into the Dürr® trolley.

Inside, patented BowTie® filter inserts handle the actual overspray capture. When filters load up, only the BowTie® inserts need to be replaced — the BLOX™ housing stays in place with most coating types.
